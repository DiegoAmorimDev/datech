

# DATECH - Site Institucional

Site moderno e impactante para a empresa de desenvolvimento web **DATECH**, com cor principal **azul claro** e animações elegantes.

## Páginas e Seções

### Landing Page
- **Hero Section** — Nome "DATECH" com animação de digitação/fade-in, tagline sobre desenvolvimento web, botão CTA "Fale Conosco" com efeito hover
- **Serviços** — Cards animados (fade-in ao scroll) para: Desenvolvimento de Sites, Sistemas Web, Landing Pages, E-commerce
- **Sobre / Diego Amorim** — Seção com foto placeholder, bio do fundador como desenvolvedor e arquiteto principal
- **Portfolio** — Grid de projetos com efeito hover scale e overlay
- **Depoimentos** — Carousel com depoimentos de clientes (dados de exemplo)
- **Contato** — Formulário de contato com validação + links para redes sociais/WhatsApp

### Layout Global
- **Navbar** fixa com logo DATECH, links suaves (scroll) e efeito de blur no background
- **Footer** com informações da empresa e links

## Design e Animações
- Paleta azul claro (#38bdf8 / sky-400) com gradientes sutis
- Animações de entrada (fade-in, scale-in) nos elementos ao aparecerem
- Efeitos hover em botões e cards
- Background com formas geométricas/gradientes animados
- Tipografia moderna e espaçamento generoso
- Totalmente responsivo (mobile-first)
